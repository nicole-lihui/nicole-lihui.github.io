# SpringBoot init


##　spring init的推荐姿势
```bash
$ git clone git@github.com:wit-course/xlab-service.git
$ cd xlab-service
$ spring init --description 'XLab Service' -a xlab-service -g com.maxwit.ooda --package-name com.maxwit.ooda.xlab -d=web,websocket,redis -x -n XLab --force

# 解析
# -x 解压
# -n 名字
# -d 依赖
```
